# ParentPulse — child development & parenting intelligence

- **Base URL:** `https://parentpulse.vercel.app`
- **Coverage:** Global. Responds in any language.
- **Summary:** Grounded answers for the decisions parents actually face — is my child developing on track, is this product recalled, which school/childcare, how do I handle sleep/eating/behavior. Structured JSON aligned to CDC/AAP milestone and safety guidance and CPSC recall data, not a model guessing about a child's health.

Call any endpoint with `bankr x402 call "<baseUrl><path>?<params>"`. Inspect first with `bankr x402 schema "<baseUrl><path>"`. Add `lang` for any language.
