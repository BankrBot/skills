# bankrnoob

**BankrBot for complete beginners. Free. No x402. No setup.**

## What is this?

BankrBot lets you create AI agents that can do things with money. 
But the docs assume you already know what "x402" and "agent skills" mean.
This skill doesn't.

## Commands

| Command | What you get |
|---------|-------------|
| `bankrnoob explain x402` | "HTTP 402 = 'Pay me first'. x402 = a way for your AI agent to pay for stuff automatically." |
| `bankrnoob explain skill` | "A skill is like a phone app for your AI agent. You install it, and your agent gains a new ability." |
| `bankrnoob explain agent` | "An agent is your AI assistant that can actually do things — not just chat." |
| `bankrnoob start` | "Step 1: Get an API key. Step 2: Pick a skill. Step 3: Run it." |

## Why

BankrBot is powerful. But its documentation assumes technical knowledge.
This skill is a friendly front door for everyone else.

## Install

```bash
bankr install bankrnoob
```
